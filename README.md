# rl

Implementations of reinforcement learning examples.

## Dependencies (tested)

- Windows 10
  - GPU Driver 471.11
  - CUDA 11.1
- Ubuntu 18.04
  - GPU Driver 470.57
  - CUDA 11.0
- Python 3.8
  - gym 0.21.0
  - numpy 1.21.5
  - pyglet 1.5.21
  - matplotlib 3.1.1
  - torch==1.9.1+cu111
  - torchvision==0.10.1+cu111
  - torchaudio==0.9.1

## Installation

    $ git clone git@github.com:takuya-ki/rl.git
    $ pip install -r requirements.txt

## Usage

    $ python cartpole/cartpole_qrl.py
